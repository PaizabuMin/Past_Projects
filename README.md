# Past_Projects

This workspace contains several course and project repositories. Below is an overview of each project folder and its main focus.

## Projects

### `ATM_Bank_Prototype`
A two-part ATM and bank system project.

- Part 1 is a team-based design and implementation of an ATM, a bank, and an init program, with a router to forward messages between ATM and bank.
- Part 2 is an individual security phase focused on attacking other teams' protocol designs.
- Teaches protocol design, specification, system architecture, security analysis, and team development.

### `Branch_Prediction`
A computer systems architecture project for simulating branch predictors.

- Implements branch prediction strategies using trace files to predict branch outcomes.
- Covers bimodal, two-level adaptive, local history (p-share), and g-share predictors.
- Includes C/C++ simulator code, test traces, and validation scripts.

### `Cryptography_Exploit`
A cryptography reverse-engineering project.

- Analyzes ciphertext streams produced by a bank protocol using a fixed-key block cipher in ECB mode.
- Focuses on inferring message formats without decrypting the data.
- Includes tasks to classify messages and perform replay or modification attacks.

### `MicroOCaml`
A programming language implementation project in Racket.

- Builds an OCaml-like language interpreter and compiler.
- Supports core language constructs, primitives, data structures, functions, rest parameters, exception handling, and I/O.
- Organized incrementally with feature milestones and tests.

### `Tomosulo`
A computer systems architecture project implementing Tomasulo's algorithm.

- Simulates a dynamic instruction scheduler with reservation stations and register alias tables.
- Processes instruction trace files and models pipelined functional units with latency.
- Uses C/C++ driver code and requires completion of the simulator core.
